# 更新日志 / Changelog

## v0.3.1（本版本）

### 修复与改进

1. **主播卡片极致紧凑**
   - 状态徽章与观看人数改为缩略图浮层，删除独立的徽章行与流地址行
   - 流地址复制功能合并为图标按钮（悬停/长按可查看地址）
   - 卡片内容区仅保留两行（用户名行 + 操作行），消除所有冗余空白
   - 桌面端网格列宽由 260px 调整为 230px，单屏容纳更多卡片

2. **录制文件表格全面适配安卓端**
   - 页面固定高度，表体区域独立双向滚动，表头吸附不丢失
   - 字号 10px、字间距收紧、内边距与复选框/徽章/进度条同步缩小
   - 起始时间在移动端显示为短格式（MM-DD HH:mm）
   - 操作列折叠为图标按钮（播放/预览/后处理/删除），功能全部保留
   - 页头副标题与批量按钮移动端自适应换行

3. **全新 UI 设计（Noir Crimson）**
   - 深色：暖调墨黑 + 绯红主色；浅色：暖调纸白 + 同系绯红，随系统自动切换
   - 更大的圆角（0.75rem）、柔和分层投影、按钮按压缩放反馈、主按钮立体质感
   - 表头微型大写标签风格、选中文本主色高亮、抗锯齿字体渲染
   - 登录页氛围光斑背景 + 品牌标识

4. **Web UI 全局安卓适配**
   - 底部导航：毛玻璃 + 激活项顶部指示条 + `env(safe-area-inset-bottom)` 安全区
   - 主内容区预留导航栏 + 安全区高度；Toast 上移至导航栏之上
   - 对话框限宽 `100vw - 2rem`；`viewport-fit=cover` 与 `theme-color` 元数据
   - 保留原项目全部功能，桌面端体验同步提升

### 部署方式（沿用预构建镜像 + 预编译二进制）

`data/bin/stripchat-recorder` 已替换为包含上述全部改动的新编译二进制（x86_64 Linux），
`docker-compose.yml` 无需改动，直接 `docker compose up -d` 即可。
